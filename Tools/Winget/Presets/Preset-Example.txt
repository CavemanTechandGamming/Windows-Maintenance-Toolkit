# Preset Example - how preset bundles work (reference only; not installed from the menu)
#
# Presets are plain-text lists of WinGet install script paths. The launcher runs them in
# order when you choose  W -> P -> [1] through [4].  Edit presets with  W -> P -> E1-E4.
#
# RULES
# -----
#   * One script path per line, relative to the Tools\ folder (see paths below).
#   * Lines starting with # are skipped - use for notes AND for apps you do not want yet.
#   * Blank lines are ignored.
#   * To ENABLE an app: remove the leading # so the line begins with Winget\
#   * To DISABLE an app: add # at the very beginning of the line.
#   * Section headers like "# --- Browsers ---" are comments only.
#   * Optional: add your own comment lines anywhere (must start with #).
#   * Do not put comments on the same line as a path - uncommented paths must be path-only.
#
# GETTING STARTED
# ---------------
#   1. Open Preset-1.txt through Preset-4.txt (or copy from this file into one of them).
#   2. Uncomment only the apps you want in that bundle (delete # at line start).
#   3. Save, then run  MaintenanceLauncher.bat  ->  W  ->  P  ->  [1-4].
#
# WORKED EXAMPLE - if these three lines were in Preset-1.txt WITHOUT a leading #,
# the launcher would install Chrome, then VLC, then 7-Zip, in that order:
#
#   Winget\Apps\Browsers\Winget-Install-Chrome.bat
#   Winget\Apps\Media\Winget-Install-VLC.bat
#   Winget\Apps\SystemTools\Winget-Install-7Zip.bat
